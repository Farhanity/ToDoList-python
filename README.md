# 📝 Консольный To-Do менеджер с авторизацией

Учебный пет-проект на Python. Позволяет управлять списком задач через терминал. Поддерживает регистрацию и вход пользователей, задачи хранятся в базе данных SQLite и не теряются после перезапуска.

## 🚀 Возможности

* **Многопользовательская система** — регистрация и авторизация с паролем
* **Управление задачами** — добавление, выполнение, просмотр текущих и завершённых задач
* **Описание к задаче** — можно добавить комментарий (через `: ` после названия)
* **Сохранение данных** — всё автоматически сохраняется в локальную БД SQLite (`todo.db`)
* **Простой консольный интерфейс** — команды на русском языке

## 🛠️ Технологии

* **Python 3.10+** (используется `match/case`)
* **SQLite3** (встроенный модуль, без внешних зависимостей)
* **Git** для контроля версий

## 📦 Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone https://github.com/Farhanity/ToDoList-python.git
cd ToDoList-python
```

2. Убедитесь, что установлен Python версии 3.10 или выше:

```bash
python --version
```

3. Запустите приложение:

```bash
python main.py
```

Никаких дополнительных библиотек устанавливать не нужно — используется только стандартная библиотека Python.

## 🎮 Как пользоваться

При первом запуске вам будет предложено:

* `1` — зарегистрироваться (придумайте логин и пароль)
* `2` — войти под существующей учётной записью
* `3` — выйти

После входа вы попадаете в главное меню.

### Доступные команды

| Команда                        | Действие                                           |
| ------------------------------ | -------------------------------------------------- |
| `Добавить Купить хлеб: ржаной` | Добавить задачу «Купить хлеб» с описанием «ржаной» |
| `Добавить Позвонить маме`      | Добавить задачу без описания                       |
| `Выполнить Купить хлеб`        | Отметить задачу как выполненную                    |
| `Вывести_Текущие`              | Показать список невыполненных задач                |
| `Вывести_Выполненные`          | Показать список выполненных задач                  |
| `Выход`                        | Выйти из приложения                                |

Все команды нечувствительны к регистру букв.

## 🗄️ База данных

Проект использует SQLite. При первом запуске автоматически создаются две таблицы:

### `users`

Хранит логины и пароли пользователей.

### `tasks`

Хранит задачи с привязкой к пользователю, статусом выполнения и описанием.

Файл базы данных `todo.db` появится в папке проекта после первого запуска.

### Классы

#### `ToDoDatabase`

Слой работы с БД:

* подключение к SQLite;
* создание таблиц;
* CRUD-запросы;
* регистрация пользователей;
* авторизация пользователей.

#### `To_Do_List`

Бизнес-логика приложения, связывает интерфейс и базу данных.

## ⚠️ Важно

* Пароли хранятся в открытом виде. Проект является учебным, поэтому не используйте данный подход в реальных системах.
* Это консольное приложение, для работы требуется терминал.
